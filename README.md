# 🚀 Admin Dashboard (React + Vite)

A modern, high-performance admin dashboard built with **React, TypeScript, and Vite**.  
Designed with scalability, performance, and clean UI/UX in mind.

## ✨ Features

- 📊 Interactive charts (Line, Bar, Pie, Area)
- ⚡ Lazy loading & code splitting
- 👀 Flex box and grid system for styling
- 🎯 Optimized performance (Lighthouse 90+)
- 🌙 Dark mode support
- 🧭 Sidebar navigation
- 📱 Fully responsive layout
- ♿ Accessibility improvements

---

## 🧠 Performance Optimizations

This project focuses heavily on performance:

- ✅ Lazy-loaded components using `React.lazy`
- ✅ Reduced CLS (Cumulative Layout Shift)
- ✅ Skeleton loaders instead of blocking UI and spinner loading for fallback the App
- ✅ Memoized components to prevent unnecessary re-renders
- ✅ Optimized bundle size & tree-shaking
